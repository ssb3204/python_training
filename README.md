## 🎯 저장소의 목표

* **알고리즘 문제 해결 능력 향상**: 꾸준한 문제 풀이를 통해 논리적 사고와 문제 해결 능력을 기릅니다.

* **Python 숙련도 향상**: 다양한 알고리즘과 자료구조를 Python으로 구현하며 언어 숙련도를 높입니다.
* **주기적인 코딩 테스트 연습**: 생각났을때 잠시하는 공부가 아닌 주기적인 공부를 통해 알고리즘(코딩 테스트)에 대한 이해도를 높인다.
